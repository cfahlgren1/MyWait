## # Tech Stack

- **Apache Kafka** _(Messaging Queue)_
- **AWS Lambda** _(Serverless Ingestion API)_
- **Node.js / Express** _(Server / Worker)_
- **Socket.io** _WebSocket Library for Realtime Communication_
- **React.js** (_Dashboard Frontend_)
